# Real Estate Frontend

This project is a modern React + Vite frontend for a real estate platform where users can browse featured properties, view detailed listing pages, and explore reviews in a clean and responsive interface. The UI has been redesigned with a premium layout style, including a polished hero section, card-based property browsing, improved property details with image gallery and video tour support, and a consistent visual system across header, footer, and content sections. The application integrates with backend APIs for fetching property data and supports smooth navigation for a user-friendly browsing experience.

The project also includes an admin workflow with secure login and a dedicated dashboard to manage property listings. Admin users can access a structured table view of all properties, delete existing entries, and create new listings from a dialog-based form that supports image and video uploads.
